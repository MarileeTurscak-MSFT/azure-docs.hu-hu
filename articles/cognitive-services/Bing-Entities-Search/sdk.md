# <a name="bing-search-sdk"></a>Bing keresési SDK
A Bing entitás keresési API-mintákat a forgatókönyvek közé tartozik, amely:
1.  Keresse meg például Tom körutazás entitást, és részletes információk.
2.  Valószínűleg több leképezések a lekérdezések kifejezések Egyértelműsítő kezelni.
3.  Keresse meg a helyi entitás, például egy étterem, és részletes információkat biztosít az beszerzése.
4.  Keresse meg például az éttermekben helyi vállalatok számára, és részletes információk.
5.  Indítható el egy hibás kérelem és a hibakezelés.

> [!NOTE] 
> Néhány SDK-k a GA, illetve dokumentációját módosítások folyamatban.
