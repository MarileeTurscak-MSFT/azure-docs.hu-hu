# <a name="iot-solution-accelerators-documentation"></a>Az IoT-megoldásgyorsítók dokumentációja

Az IoT-megoldásgyorsítók gyorsan használatba vehetők az IoT-megoldások létrehozásához. Az új eszközök hozzáadását és a meglévő eszközök csatlakoztatását eszközspecifikus SDK-k segítségével végezheti, amelyek több platformon, így Linuxon, Windowson és valós idejű operációs rendszereken is használhatók. Rugalmas méretezést valósíthat meg, akár mindössze néhány érzékelőről, akár több millió egyszerre kapcsolódó eszközről van szó, és kihasználhatja az Azure globális elérhetőségét &mdash; a projekt méretétől függetlenül.
