# <a name="gtm-benefits-on-the-marketplace"></a>A piactéren GTM előnyei   
A piactéren kirakatokkal az új listaelem kerülnek szabad piactér GTM előnye kap. Után átjáróra a listában, a Microsoft értékesítési szakemberek kapcsolatba lépjünk Önnel vagy gyorsan létrehozhatja a marketing névjegy tevékenységeit. Nincs szükség ahhoz, hogy kivéve, ha próbál kapcsolódni a Microsoft bevonásához.  

A Microsoft által biztosított tevékenységek eltérőek lehetnek attól függően, hogy a megoldás állapota a piactéren. Előnyöket jelentősen csökkenti a listaelemek.  

| kirakat | Rendelkezésre álló további előnyökkel is jár  |  
|:--- |:--- |  
| AppSource | <table> <tr><td>Microsoft Gold partnerek</td></tr> <tr><td>Próba alkalmazások</td></tr> </table> |  
| Azure Piactér | <table> <tr><td>Transact ajánlatok</td></tr> <tr><td>Próbaajánlatokra</td></tr> </table> |  

Ezek a tevékenységek hatásának maximalizálása érdekében a Microsoft javasolja, hogy az indítási terv megvalósításához készen van. Előfordulhat, hogy szeretné használni a kezdőlapja a indítási taktikai számos.
