# <a name="supported-browsers-and-devices-for-the-azure-portal"></a>Támogatott böngészők és az eszközöknek az Azure-portálon
Futtathatja a [Azure-portálon](https://portal.azure.com) minden modern asztalon, tábla-eszközöket, és a böngészők.

### <a name="supported-devices"></a>Támogatott eszközök
Az Azure portálon futó kiváló modern számítógépek, Mac számítógépek és táblagépek. Ha szeretné az Azure erőforrások kezeléséhez egy mobileszközről, próbálja meg az Azure mobile Apps iPhone-és Android rendszeren elérhető. További információkért lásd: [Azure mobile Apps](https://azure.microsoft.com/features/azure-portal/mobile-app/).

### <a name="supported-browsers"></a>Támogatott böngészők
Azt javasoljuk, hogy az operációs rendszer kompatibilis legfrissebb böngészőt használ. Az alábbi böngészők támogatják:

* A Microsoft Edge (a legújabb verzió)
* Internet Explorer 11
* Safari (csak a Mac, legújabb verzió)
* Chrome (a legújabb verzió)
* Firefox (a legújabb verzió)
