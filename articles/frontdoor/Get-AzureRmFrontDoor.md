# Get-AzureRmFrontDoor
Bejárati ajtajának konfigurációjának lekérése

## SZINTAXIS

```
Get-AzureRmFrontDoor [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## LEÍRÁS
A **Get-AzureRmFrontDoor** cmdletGet lekérdezi az aktuális előfizetésben az összes meglévő bejárati ajtó egyezések a megadott adatokat.
