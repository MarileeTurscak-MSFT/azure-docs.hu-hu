# <a name="how-to-configure-an-application-proxy-application-to-use-pingaccess"></a>Az alkalmazásproxy alkalmazás PingAccess konfigurálása

Az együttműködés most a PingAccess jogosíthatja alkalmazásproxy fejléc-alapú hitelesítést használó alkalmazások teszi lehetővé. Ha az alkalmazások nem a fejlécek, tekintse meg a [egyszeri bejelentkezés dokumentáció](manage-apps/application-proxy-configure-single-sign-on-with-kcd.md) egyéb beállítások leírását.

## <a name="overview-of-steps-and-recommended-documents"></a>A lépéseket és a javasolt dokumentumok áttekintése

Az alkalmazás konfigurálása PingAccess, négy lépésben történik:

1.  Konfigurálja az Application Proxy összekötőket

2.  Az Azure AD Application Proxy alkalmazás létrehozása

3.  Töltse le & PingAccess konfigurálása

4.  Alkalmazások konfigurálása az PingAccess

A lépések a részletekért lásd: a [egyszeri bejelentkezés fejlécek dokumentációját](manage-apps/application-proxy-configure-single-sign-on-with-ping-access.md).
