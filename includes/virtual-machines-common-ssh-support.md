## <a name="supported-ssh-key-formats"></a>Támogatott SSH-kulcs formátumok

Az Azure jelenleg támogatja az SSH protokoll 2 (SSH-2) RSA nyilvános-titkos kulcspárok 2048 bites minimális hosszal. Például ED25519 és ECDSA más kulcs formátum nem támogatott.
