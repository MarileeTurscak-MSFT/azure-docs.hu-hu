# <a name="how-to-configure-an-application-proxy-application-to-use-pingaccess"></a>Az alkalmazásproxy alkalmazás PingAccess konfigurálása

Az noaa Ügynökséggel PingAccess mostantól lehetővé teszi az előnyöket, az Application Proxy fejlécalapú hitelesítést használó alkalmazások. Ha az alkalmazások nem használja a fejlécek, tekintse meg a [egyszeri bejelentkezés dokumentáció](manage-apps/application-proxy-configure-single-sign-on-with-kcd.md) más lehetőség részletes.

## <a name="overview-of-steps-and-recommended-documents"></a>Lépéseit és ajánlott dokumentumok áttekintése

Alkalmazás konfigurálása a pingaccess segítségével, négy lépésben történik:

1.  Alkalmazásproxy-összekötők konfigurálása

2.  Egy Azure AD alkalmazásproxy-alkalmazás létrehozása

3.  Töltse le, és a PingAccess konfigurálása

4.  Konfigurálhatja az alkalmazásokat a PingAccess

A lépések részletes ismertetéséért tekintse meg a [egyszeri bejelentkezést a fejlécek dokumentációjában](manage-apps/application-proxy-configure-single-sign-on-with-ping-access.md).
