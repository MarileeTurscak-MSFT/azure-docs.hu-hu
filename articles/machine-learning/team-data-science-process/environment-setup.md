# <a name="set-up-data-science-environments-for-use-in-the-team-data-science-process"></a>Adatelemzési környezetek beállítása a csoportos adatelemzési folyamatban történő felhasználásra
Az Team tudományos folyamat különböző adatok tudományos környezetekben a tárolás, az adatfeldolgozás és az adatok elemzése használ. Azure Blob Storage számos különböző Azure virtuális gépeken, a HDInsight (Hadoop) fürtök és az Azure Machine Learning munkaterületek tartalmaznak. A döntést, mely a környezetben használandó függ modellezni típusát és adatok mennyisége és az adatok a felhőben a célhelyre. 

* Ennek eldöntése során figyelembe kell venni kérdésekben útmutatóért lásd: [tervezze meg a Azure Machine Learning adatok tudományos környezet](plan-your-environment.md). 
* Katalógus néhány speciális elemzési során esetleg felmerülő forgatókönyv, lásd: [az Team tudományos folyamat forgatókönyvei](plan-sample-scenarios.md)

Ez a menüben a témakörök, amelyek bemutatják, hogyan állíthatja be a különböző adatok tudományos környezetekben az Team tudományos folyamat által használt mutató hivatkozásokat tartalmaz.

A **Microsoft adatok tudományos virtuális gép (DSVM)** Azure virtuális gép (VM) képként érhetők el. A virtuális gép előtelepített és konfigurált számos népszerű eszköz adatelemzés és a gépi tanulás általánosan használt. A DSVM Windows és Linux érhető el. További információkért lásd: [a Linux és a Windows számára a felhőalapú adatokat tudományos virtuális gép bemutatása](../data-science-virtual-machine/overview.md).
