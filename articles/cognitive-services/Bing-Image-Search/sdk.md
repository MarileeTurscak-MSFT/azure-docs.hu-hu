# <a name="bing-search-sdk"></a>Bing keresési SDK
A Bing kép keresési API-mintákat senarios tartalmazza, amelyek:
1.  A lekérdezés egy egyszerű lemezképhez.
2.  Keresés adott típusú és méretarányoknak megfelelően ábrázoló kép szűrési lehetőségek való korlátozása.
3.  Jelenleg a különféle közösségi hálózatokon trendek képek keresése.
4.  Lekérdezés gazdag részletes kép, például a forrás lemezkép, reprezentatív lekérdezésszöveg, stb.

Bing keresési SDK-k ellenőrizze webes keresési funkcióját könnyen hozzáférhető a következő programozási nyelven:
* Ismerkedés a [.NET minták](https://github.com/Azure-Samples/cognitive-services-dotnet-sdk-samples/tree/master/BingSearchv7)
    * [NuGet-csomag](https://www.nuget.org/packages/Microsoft.Azure.CognitiveServices.Search.ImageSearch/1.2.0)
    * Lásd még: [.NET-kódtárakra](https://github.com/Azure/azure-sdk-for-net/tree/psSdkJson6/src/SDKs/CognitiveServices/dataPlane/Search/BingImageSearch) definíciókat és függőségeit.
* Ismerkedés a [Node.js – minták](https://github.com/Azure-Samples/cognitive-services-node-sdk-samples) 
    * Lásd még: [Node.js szalagtárak](https://github.com/Azure/azure-sdk-for-node/tree/master/lib/services/imageSearch) definíciókat és függőségeit.
* Ismerkedés a [Java-minták](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples) 
    * Lásd még: [Java szalagtárak](https://github.com/Azure/azure-sdk-for-java/tree/master/azure-cognitiveservices/search/bingimagesearch) definíciókat és függőségeit.
* Ismerkedés a [Python-minták](https://github.com/Azure-Samples/cognitive-services-python-sdk-samples) 
    * Lásd még: [Python szalagtárak](https://github.com/Azure/azure-sdk-for-python/tree/master/azure-cognitiveservices-search-imagesearch) definíciókat és függőségeit.

Az egyes nyelvekhez SDK-minták Előfeltételek és a minták telepítése/fut fontos fájljai közé tartozik.
