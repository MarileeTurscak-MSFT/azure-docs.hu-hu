> [!IMPORTANT] 
> Azure az erőforrások létrehozására és kezelésére két különböző üzembe helyezési modellel rendelkezik: [Resource Manager és klasszikus](../../../resource-manager-deployment-model.md). Ez a cikk a klasszikus telepítési modell használatát bemutatja. A Microsoft azt javasolja, hogy az új telepítések esetén a Resource Manager modellt használja. Virtuálisgép-bővítmények az erőforrás-kezelő modellel kapcsolatos további információkért lásd: [Itt](../extensions-features.md?toc=%2fazure%2fvirtual-machines%2fwindows%2ftoc.json).
