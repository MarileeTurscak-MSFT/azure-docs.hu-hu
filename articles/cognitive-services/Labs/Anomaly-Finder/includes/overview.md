## <a name="overview"></a>Áttekintés
Az Anomáliadetektálási kereső API lehetővé teszi-adatok figyelése adott idő alatt, és a machine Learning segítségével úgy, hogy a jobb oldali statisztikai modell automatikusan alkalmazza az egyedi adatait alkotási rendellenességek észlelését. Ez lehetővé teszi a szervezetek számára, az adatminőségi karbantartása, megbízható szolgáltatásokat, üzleti incidensek azonosítása és pontosítsa az üzleti megközelítés rendellenes adatok manuális azonosítási nehéz problémák elleni nélkül.
