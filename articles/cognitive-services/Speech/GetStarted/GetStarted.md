# <a name="get-started-with-speech-service"></a>Ismerkedés a beszédfelismerés szolgáltatás

A Microsoft beszéd felismerési szolgáltatás segítségével hang szöveggé alakítani, tekintse meg a beszédfelismerés [REST API-k](GetStartedREST.md) vagy a [klienskódtárak](GetStartedClientLibraries.md).

A Microsoft szöveget beszéddé átalakító szolgáltatás használatához, tekintse meg a [szöveg-hang transzformációs API-referencia](../api-reference-rest/bingvoiceoutput.md).
