# <a name="project-url-preview-use-and-display-requirements"></a>Projekt URL-cím Preview használja, és megjeleníti a követelmények

Csak a projekt URL-cím Preview adatait használatával megjelenítheti preview kódtöredékek és hiperhivatkozással ellátott miniatűr képeket a forráshelyeket a végfelhasználó által kezdeményezett URL-cím osztozik a közösségi média, csevegési botot vagy hasonló ajánlatok. Meg kell nem másolása, tárolni vagy projekt URL-cím Preview származó adatok gyorsítótárazása. Tiltsa le az előzetes verziójú funkciók, amelyek jelenhet meg a webhely vagy a tartalmak tulajdonosai érkező kérésekre kell fogadják el.
