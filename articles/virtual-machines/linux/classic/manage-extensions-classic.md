# <a name="manage-virtual-machine-extensions"></a>Virtuálisgép-bővítmények kezelése
Útmutatás található, hozzáadása, módosítása, vagy távolítsa el a Virtuálisgép-bővítmények a Windows vagy Linux rendszerű virtuális gépek Azure-on.

> [!IMPORTANT] 
> Azure az erőforrások létrehozására és kezelésére két különböző üzembe helyezési modellel rendelkezik: [Resource Manager és klasszikus](../../../resource-manager-deployment-model.md). Ez a cikk a klasszikus telepítési modell használatát bemutatja. A Microsoft azt javasolja, hogy az új telepítések esetén a Resource Manager modellt használja. A Resource Manager modellt Virtuálisgép-bővítmények használatával kapcsolatos információkért lásd: [Itt](../extensions-features.md?toc=%2fazure%2fvirtual-machines%2flinux%2ftoc.json).
