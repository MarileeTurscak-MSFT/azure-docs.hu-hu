# <a name="upload-images"></a>Képek feltöltése #

A felhasználói felület megismeréséhez egyszerre legfeljebb 2 MB méretű fájl minden egyes kép legfeljebb öt képek feltöltése. Győződjön meg arról, ellenőrizze a fájlméret feltöltés előtt.

![Képek feltöltése](images/1-try-image-1.png)

## <a name="submit-for-moderation"></a>Küldje el a moderálás ##
Ha egyszerre több mint öt képek feltöltése fel, a Trashcan ikon segítségével távolítsa el a nem kívánt lemezképet. Majd küldje el az automatizált moderálás a lemezképeket.

![Küldje el a lemezképek](images/1-try-image-2.png)
