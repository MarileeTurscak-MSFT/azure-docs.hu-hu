## <a name="supported-ssh-key-formats"></a>Támogatott SSH-kulcs formátumok

Az Azure jelenleg támogatja az SSH protokoll 2 (SSH-2) az RSA nyilvános-titkos kulcspárok a 2048 bites minimális hosszát. Például ED25519 és ECDSA más kulcs formátum nem támogatott.
