Az Azure Cosmos DB-kötések csak az SQL API-val használhatók. A többi Azure Cosmos DB API, például a [MongoDB API](../articles/cosmos-db/mongodb-introduction.md), a [Cassandra API](../articles/cosmos-db/cassandra-introduction.md), a [Gremlin API](../articles/cosmos-db/graph-introduction.md) és a [Table API](../articles/cosmos-db/table-introduction.md) esetén az API statikus ügyfelével érheti el az adatbázist a függvényből.
