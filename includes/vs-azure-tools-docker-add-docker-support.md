1. A Visual Studio **Megoldáskezelőben**, kattintson a jobb gombbal a projektre, és válassza ki **Hozzáadás > Docker-támogatás** a helyi menüből.
   
    ![Adja hozzá a helyi menü Docker-támogatás](media/vs-azure-tools-docker-add-docker-support/docker-support-context-menu.png)
2. Docker-támogatás hozzáadása az ASP.NET Core web projektet eredményez a projektet, beleértve a docker-fájlban, a központi telepítés Windows PowerShell-parancsfájlok és a Docker tulajdonság fájlokat ad hozzá több Docker kapcsolatos fájlt is. 
   
    ![Docker-fájlok hozzáadása a projekthez](media/vs-azure-tools-docker-add-docker-support/docker-files-added.png)
