Az [Azure Cosmos DB-t kipróbálhatja ingyenesen](https://azure.microsoft.com/try/cosmosdb/), Azure-előfizetés, díjfizetés és elköteleződés nélkül. Másik lehetőségként használhatja az [Azure Cosmos DB Emulatort](https://docs.microsoft.com/azure/cosmos-db/local-emulator) a következő URI-val: https://localhost:8081. Az elsődleges kulcs a [Kérelmek hitelesítése](../articles/cosmos-db/local-emulator.md#authenticating-requests) című részben található.
