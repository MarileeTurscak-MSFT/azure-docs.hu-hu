# <a name="get-started-with-the-projectonline-connector"></a>A ProjectOnline összekötő az első lépései
A Project Online a projekt portfóliót kezeléséhez (a PPM) és a mindennapos munka a Microsoft online rugalmas megoldást. Office 365 keresztül továbbítva, Project Online lehetővé teszi a szervezetek gyorsan használatba hatékony projekt felügyeleti lehetőségeket a megtervezését, rangsorolhatja és projektek és a projekt portfóliót beruházások kezelése – szinte bárhonnan szinte bármilyen eszközről.

Most hozzon létre egy Logic app kezdheti, lásd: [logikai alkalmazás létrehozása](../logic-apps/quickstart-create-first-logic-app-workflow.md).
