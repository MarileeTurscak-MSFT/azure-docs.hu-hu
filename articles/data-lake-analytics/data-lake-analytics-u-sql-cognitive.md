# <a name="get-started-with-the-cognitive-capabilities-of-u-sql"></a>Ismerkedés a U-SQL kognitív képességei

## <a name="overview"></a>Áttekintés
U-SQL kognitív képességei lehetővé teszi a fejlesztőknek, a big data jellegű programok put intelligencia használatára. 

A következő kognitív képességek érhetők el:
* Lemezképkészítésre: Arcfelismerés [minta](https://github.com/Azure-Samples/usql-cognitive-imaging-ocr-hello-world)
* Képfeldolgozás: Észlelése emotion [minta](https://github.com/Azure-Samples/usql-cognitive-imaging-emotion-detection-hello-world)
* Képfeldolgozás: (Címkézés) objektumok észlelése [minta](https://github.com/Azure-Samples/usql-cognitive-imaging-object-tagging-hello-world)
* Képfeldolgozás: OCR (optikai karakterfelismerés) [minta](https://github.com/Azure-Samples/usql-cognitive-imaging-ocr-hello-world)
* Szöveg: Kulcsfontosságú kifejezések kinyerése és Hangulatelemzés [minta](https://github.com/Azure-Samples/usql-cognitive-text-hello-world)
