# <a name="prepare-data-with-the-azure-machine-learning-data-prep-sdk-for-python"></a>Készítse elő adatait az Azure Machine Learning Data Prep SDK Pythonhoz készült
 
Adat-előkészítés a machine learning munkafolyamat fontos része. A modellek több pontos és hatékony, ha ők is hozzáférhetnek, amely könnyebben felhasználása formátumú adatokat megtisztítani lesz. Használhatja a [az Azure Machine Learning Data Prep SDK](https://docs.microsoft.com/python/api/overview/azure/dataprep?view=azure-dataprep-py) betölteni az adatokat a különböző formátumok, átalakítja a további használatához és az adatokat írni az eléréséhez a modellek helyét.

1. [Adatok betöltése](how-to-load-data.md )
2. [Adatok átalakítása](how-to-transform-data.md )
3. [Adatok írása](how-to-write-data.md )
