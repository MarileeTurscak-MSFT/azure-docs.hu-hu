# <a name="how-to-select-permissions-for-a-given-api"></a>Egy adott API-engedélyek kiválasztása

A hitelesítési végpont található az alkalmazás a [Azure-portálon](https://portal.azure.com).

-   Lépjen az [Azure Portalra](https://portal.azure.com).

-   Kattintson a bal oldali navigációs ablak **Azure Active Directory**.

-   Kattintson a **App regisztrációk** válassza **végpontok**.

-   Nyissa meg ezt a **végpontok** lapon, amely a bérlő a hitelesítési végpontok listájában.

-   Használja a végpontot, amely jellemző a hitelesítési protokollt használ, a hitelesítési létrehozható az alkalmazás azonosítójával együtt kérés adott alkalmazásra.
