## <a name="azure-certified-program"></a>Az Azure Certified program  
Az Azure piactéren közzétett összes virtuális gép (VM) ellenőrzi az Azure hitelesített program.  
*   További tudnivalókért keresse fel a Microsoft Azure hitelesített lapon található [azure.microsoft.com/marketplace/programs/certified](https://azure.microsoft.com/marketplace/programs/certified). 

A hitelesítésszolgáltató program a következő funkciókkal rendelkezik.  
*   A virtuális gép esetén az Azure platformon és a piactér modellt kínál.  
*   A virtuális gép online lemezkép biztonsági megfelelőség szempontjából, beleértve a vírusok és kártevő lett tesztelve.  
*   A virtuális gép tartalmazza az ajánlat szintjén badging. A vállalati ügyfelek a Microsoft promóciós érvényesített megoldásként badging javítja.

### <a name="trial"></a>Próbaverzió  
Biztosít egy próbaverziója növeli az ügyfelek számára megadott engagement szintjét. Egy nagyobb engagement szintet gazdagabb lehetősége, hogy a megoldás okoz. A próba ajánlattípus lehetővé teszi az ügyfelek a megoldás felfedezése megvásárlása előtt. A próbameghívó listaelem típusa vannak a kirakatokkal előléptetés magasabb esélyét. Az ügyfél kapcsolattartás során több és több funkcióval érdeklődők is számíthat.  

Az alkalmazás vagy a listaelem típusa próbaverzió használatával ajánlat a rendszer a kísérleti környezetben, az Azure-előfizetés vagy mindkettőt. Listaelem egyéb volna telepíthető a környezetben vagy az Azure-előfizetés az ügyfél. Az alkalmazás vagy az ajánlat vásárolhat további követelmények nélkül vezetett ügyfél kell lennie. Az alkalmazás vagy az ajánlat kell minimális, ha van ilyen, végezze el a használati eset további beállításra. Az alkalmazás vagy az ajánlat tartalmaznia kell szabad támogatási legalább a próbaidőszak alatt. Az ügyfelek nurtured legyen, és figyeli a legjobb eredmények elérése érdekében a szándékos értékelési útvonalon. Hosszúan lett megadva a piactéren, és a saját alkalmazásbeli eszközintelligencia érdeklődők használja. A tippek használandó figyelésére és az ügyfelek kezelésére.
