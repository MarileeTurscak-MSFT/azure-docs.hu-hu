# <a name="sfctl"></a>sfctl
Service Fabric-fürtök és entitások felügyeletére szolgáló parancsokat. A Service Fabric 6.2 futásidejű kompatibilis verziója.

Parancsok főnév-verb mintát követnie. További információkért lásd: alcsoportokat.

## <a name="subgroups"></a>Alcsoportok
|Alcsoport|Leírás|
| --- | --- |
| [Alkalmazás](service-fabric-sfctl-application.md) | Létrehozása, törlése és alkalmazásokat és típusok. |
| [Chaos](service-fabric-sfctl-chaos.md) | Indítás, Leállítás, és a chaos teszt szolgáltatás jelentést. |
| [cluster](service-fabric-sfctl-cluster.md) | Válassza ki, kezelése és a Service Fabric-fürtök működik. |
| [Összeállítása](service-fabric-sfctl-compose.md) | Létrehozása, törlése és kezelése a Docker Compose alkalmazások. |
| [Tároló](service-fabric-sfctl-container.md) | Tároló kapcsolatos parancsok futtatnak egy fürt csomópontja. |
| [van](service-fabric-sfctl-is.md) | Lekérdezi és parancsainak elküldését az infrastruktúra-szolgáltatás. |
| [Csomópont](service-fabric-sfctl-node.md) | A csomópontok, amely egy fürt része legyen kezelése. |
| [partition](service-fabric-sfctl-partition.md) | Lekérdezi és partíciók valamelyik szolgáltatás kezelése. |
| [Tulajdonság](service-fabric-sfctl-property.md) | A Service Fabric neve tároló és a lekérdezési tulajdonságok. |
| [replica](service-fabric-sfctl-replica.md) | A szolgáltatáspartíciók tartozó replikák kezelése. |
| [rpm](service-fabric-sfctl-rpm.md) | Lekérdezi és parancsainak elküldését a repair-kezelő szolgáltatás. |
| [a fürt rendszergazdai (SA)](service-fabric-sfctl-sa-cluster.md) | Különálló Service Fabric-fürtök kezelése. |
| [service](service-fabric-sfctl-service.md) | Létrehozása, törlése, és szolgáltatás, a típusú szolgáltatásokat és a service-csomagok kezelése. |
| [store](service-fabric-sfctl-store.md) | Alapszintű fájl szint a végre műveleteket a fürt lemezképtárolóhoz. |
