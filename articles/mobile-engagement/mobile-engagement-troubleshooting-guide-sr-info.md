# <a name="service-request-sr-information"></a>Szolgáltatáskérés (SR) adatai
> [!IMPORTANT]
> Az Azure Mobile Engagement kivonja a 3/31/2018. Ezen a lapon hamarosan után törlődni fog.
> 

A Mobile Engagement szolgáltatáshoz szolgáltatáskérés megnyitásakor, adja meg a következő információkat:

## <a name="ids-applicable-identifiers-related-to-your-issue"></a>Azonosítók: Alkalmazható azonosítók a problémával kapcsolatos:
* Alkalmazásazonosító
* A kampány Azonosítóját
* Eszközazonosító
* Felhasználói azonosító
* Felhasználónév
* Alkalmazásadatok címke

## <a name="errors-applicable-error-information"></a>Hibák: Alkalmazandó hibainformációk
* Ha a probléma akkor fordul elő API-t vagy a felhasználói felület szakasz nevével:
* Bármilyen hibaüzenet szövege
* Bármely végrehajtását követően a tesztek eredményét a [hibaelhárítási útmutatók](http://go.microsoft.com/fwlink/?LinkId=524382)

## <a name="code-applicable-coding-information"></a>Kódot: Megfelelő kódolási adatokat
* SDK-verzió és platform az alkalmazás (Android SDK 2.4.1, iOS 1.16.2 stb.)
* Letöltési hely az éles alkalmazás (vagy a fejlesztői alkalmazás APK/TGZ fájljai)
* "AndroidManifest.xml" és/vagy bármely kódrészletet az alkalmazásból az Azure Mobile Engagement kapcsolódik (a speciális hibaelhárítás)
