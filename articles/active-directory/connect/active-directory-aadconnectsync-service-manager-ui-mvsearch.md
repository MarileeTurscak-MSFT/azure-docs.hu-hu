# <a name="sync-service-manager-metaverse-search"></a>Szinkronizálás a Service Manager Metaverzum-keresés

![Szinkronizálás a Service Manager](./media/active-directory-aadconnectsync-service-manager-ui/mvsearch.png)

A metaverzum-keresés lapon az adatok kapcsolatos hibaelhárítás esetén hasznosak. A felső fele a lekérdezés attribútumok kombinációja alapján is létrehozhat. Ha elégedett a lekérdezés, kattintson **keresési**. Az alsó rácsban látható eredménye. Kiválaszthatja az oszlopokat kell lennie a látható **oszlop beállításai**.

A keresési eredmények között, válasszon ki egy objektumot, és **tulajdonságok** megtekintéséhez a [metaverzum-objektum tulajdonságai](active-directory-aadconnectsync-troubleshoot-object-not-syncing.md#metaverse-object-properties).
