# <a name="rest-api-specification-for-azure-front-door-service"></a>Azure bejárati ajtajának szolgáltatás REST API-specifikáció
Az Azure bejárati ajtajának szolgáltatás lehetővé teszi definiálása, kezelheti és figyelheti a webes forgalom globális útválasztását optimalizálásával a lehető legjobb teljesítményt és a magas rendelkezésre állású globális azonnali feladatátvételt. A bejárati ajtó alakíthatja át a globális (többrégiós) fogyasztói és a vállalati alkalmazások nagy teljesítményű, nagy teljesítményű személyre szabott modern alkalmazások, az API-k és a tartalmakat, amelyekkel globális léptékben az Azure-ral.


Az alábbi táblázat a REST API-hoz egy hivatkozást kínál Azure bejárati ajtajának szolgáltatás.

API-verzió: 2018-08-01

| A művelet csoport | Leírás |
|-------------|------------|
| [Bejárati ajtajának](https://github.com/Azure/azure-rest-api-specs/blob/master/specification/frontdoor/resource-manager/Microsoft.Network/preview/2018-08-01-preview/frontdoor.json) | Határozza meg, hogyan hozzon létre, és a bejárati ajtajának konfigurációjának kezelése |
| [Webes alkalmazás tűzfal-házirend](https://github.com/Azure/azure-rest-api-specs/blob/master/specification/frontdoor/resource-manager/Microsoft.Network/preview/2018-08-01-preview/webapplicationfirewall.json) | Meghatározza a számára a bejárati ajtó a webes alkalmazás tűzfal-házirend kezelése |
