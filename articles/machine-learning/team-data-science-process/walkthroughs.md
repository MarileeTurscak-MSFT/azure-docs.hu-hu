# <a name="walkthroughs-executing-the-team-data-science-process"></a>A forgatókönyvek az Team tudományos folyamat végrehajtása

Ezek **végpont forgatókönyvek** bemutatják az Team tudományos folyamat az adott forgatókönyv lépéseit. Azok bemutatják, hogyan kombinálhatók felhő, a helyszíni eszközök és szolgáltatások létrehozásához folyamat vagy munkafolyamat egy **intelligens alkalmazás**. A forgatókönyvek szerint vannak csoportosítva **platform** használnak.


## <a name="walkthrough-descriptions"></a>Forgatókönyv leírása

Az alábbiakban az egyes platformokon milyen forgatókönyv példákban adjon meg rövid leírása:


- [Az Azure Machine Learning-munkaterület](walkthroughs-aml-with-tdsp.md) ezen forgatókönyv oktatóanyagok és minták létrehozását mutatják be strukturált Team Adattudomány projektek az Azure Machine Learning munkaterület természetes nyelvű feldolgozási és besorolás.
- [HDInsight Spark forgatókönyvek PySpark és Scala használatával](walkthroughs-spark.md) ezek a forgatókönyvek PySpark és használjuk Scala egy Azure Spark-fürtön a prediktív elemzés.
- [HDInsight Hadoop forgatókönyvek Hive eszközzel](walkthroughs-hdinsight-hadoop.md) ezek a forgatókönyvek a Hive használata a HDInsight Hadoop-fürt a prediktív elemzés végrehajtásához.
- [Azure Data Lake forgatókönyvek U-SQL használatával](walkthroughs-azure-data-lake.md) ezek a forgatókönyvek használata U-SQL Azure Data Lake a prediktív elemzés végrehajtásához.
- [SQL Server](walkthroughs-sql-server.md) ezek a forgatókönyvek használata az SQL Server, SQL Server R szolgáltatások és az SQL Server Python Services a prediktív elemzés végrehajtásához.
- [Az SQL Data Warehouse](walkthroughs-sql-data-warehouse.md) ezek a forgatókönyvek a prediktív elemzés ehhez az SQL Data Warehouse használható. 



## <a name="next-steps"></a>További lépések

A fő összetevőből épül fel az Team tudományos folyamat tárgyalását lásd: [Team adatok tudományos folyamat áttekintése](overview.md).

A csoport az tudományos folyamata életciklussal, amely segítségével az adatok tudományos projektek szerkezeti tárgyalását lásd: [Team adatok tudományos folyamat életciklus](lifecycle.md). Az életciklus kezdetétől a végéig, projektek általában követve azok végrehajtásakor, lépéseit mutatja be. 

Témakörök, amelyek végigvezetik a feladatokat az Azure-ban tudományos folyamat alkotó áttekintését lásd: [adatok tudományos folyamat](http://aka.ms/datascienceprocess).
